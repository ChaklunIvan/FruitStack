What is FruitStack?

This is a small pet project about fruits. Made for educational purposes only. To study the work of a client-server web application and understand how to work with Api`s.
Using technologies such as:

Server -

.Net


Api`s -

Fruityvice (https://www.fruityvice.com), Unsplash (https://unsplash.com)


Client -

Angular, Bootstrap

I will leave all the source code and documentation for it on my github.
(https://github.com/ChaklunIvan/FruitStack)
